# After9 Theme Memory (Combined Reference System)

This document preserves the exact style direction used for After9 so future updates stay consistent.

## Core Blend

- Apple product pages: cinematic dark presentation, large confident type, generous spacing, premium motion.
- Resend: modern developer-polish dark UI, soft glows, clean card layering, focused CTA rhythm.
- Palantir: enterprise-grade clarity, high-trust copy tone, authority-first section structure.

## Visual Rules

- Background: near-black base with subtle gradients, mesh texture, and restrained ambient orbs.
- Typography: large headlines, high contrast, short direct lines, clear information hierarchy.
- Components: rounded-xl or rounded-full shells, thin slate borders, glass-like panels, minimal noise.
- CTA behavior: one dominant blue action, one secondary dark outline action, hover lift and glow.
- Motion: tasteful fade/float transitions only; avoid noisy or playful animation styles.

## Product Psychology Rules

- Lead with authority and trust before details.
- Use concise proof points (ratings, conversion lift, setup speed).
- Keep dense information in digestible cards.
- Reduce cognitive load in every section through clear grouping and spacing.

## UI Keywords for Future Prompts

- "Cinematic dark SaaS"
- "Premium trust-first conversion design"
- "Apple + Resend + Palantir blend"
- "Minimal but powerful"
- "Professional, not party-flyer aesthetic"
